## 🚀 快速开始

| 步骤 | 命令 |
|------|------|
| 1. 安装 mise | `curl https://mise.run \| sh` |
| 2. 激活 Shell | `echo 'eval "$(~/.local/bin/mise activate bash)"' >> ~/.bashrc` |
| 3. 设置版本 | `mise use --global node@20 python@3.12` |
| 4. 验证安装 | `node --version` / `python --version` |

> 详细安装指南和多平台说明请阅读 [快速入门](./02-快速入门.md)。
