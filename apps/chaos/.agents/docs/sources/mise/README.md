# mise Raw Sources

该目录用于存放 mise 官方资料摘录、原始技术调研记录和整理前的工作底稿。

## 使用规则

- 原始资料不作为 agent 默认优先阅读入口
- 日常查询 mise 概念、命令、配置和迁移方案时，优先阅读 [`../../references/mise/`](../../references/mise/)
- 需要追溯调研背景、版本上下文或未完全提炼的技术细节时，再阅读本目录
- 当原始资料被整理成稳定知识后，应在 `../../references/mise/` 中维护精炼页面
