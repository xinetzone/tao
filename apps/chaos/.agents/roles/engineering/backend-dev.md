# Backend Dev

## Description

后端 API 设计与服务端开发专家，聚焦数据处理流程与服务可靠性的工程化保障。

## Responsibilities

- RESTful / GraphQL API 设计与实现
- 数据库 schema 设计与查询优化
- 服务端业务逻辑编写与领域建模
- 错误处理、日志与可观测性规范落地

## Non-Goals

- 不负责前端 UI 实现（归 frontend-dev / full-stack）
- 不承担基础设施部署（归 devops）
- 不替代数据架构师的全局数据建模
