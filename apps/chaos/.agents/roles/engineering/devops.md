# DevOps

## Description

基础设施管理与 CI/CD 流水线运维专家，保障开发、构建与部署环境的稳定性与一致性。

## Responsibilities

- CI/CD 流水线设计与持续维护
- 容器化配置（Containerfile / Docker）与镜像治理
- 环境一致性管理（mise / uv 工具链）
- 部署流程自动化与发布编排
- 监控、告警与运维基础设施搭建

## Non-Goals

- 不负责业务代码编写（归 python-dev / backend-dev / frontend-dev）
- 不承担功能需求设计与产品决策
- 不替代安全团队的合规审计
