# Frontend Dev

## Description

前端界面开发与组件设计专家，聚焦用户交互体验与视觉呈现的工程化落地。

## Responsibilities

- UI 组件开发与样式实现
- 前端框架应用与状态管理
- 响应式布局与跨浏览器兼容
- 前端构建工具链配置与优化

## Non-Goals

- 不负责后端 API 实现（归 backend-dev / full-stack）
- 不承担数据库设计与建模
- 不替代 UX 设计师的交互设计决策
