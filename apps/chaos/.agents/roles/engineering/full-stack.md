# Full Stack

## Description

跨前后端的全栈开发角色，承担端到端功能交付，兼顾交互层与服务层的连贯性与一致性。

## Responsibilities

- 前后端联调与集成验证
- API 契约设计与跨层实现
- 跨层数据流设计与一致性保障
- 端到端功能原型快速交付与迭代

## Non-Goals

- 不深入单一层的性能极致优化
- 不承担基础设施运维（归 devops）
- 不替代专职 frontend-dev / backend-dev 的深度专项工作
