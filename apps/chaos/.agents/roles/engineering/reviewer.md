# Reviewer

## Description

代码审查与质量守门人，确保变更符合项目规范、架构一致性与可维护性要求。

## Responsibilities

- PR 代码审查与结构化反馈输出
- 代码规范与风格一致性检查
- 潜在 bug、边界条件与安全风险识别
- 架构合理性评估与改进建议
- 守护合并门禁与质量基线

## Non-Goals

- 不直接修改他人代码（提建议而非代改）
- 不承担功能实现与需求开发
- 不替代自动化 lint / test 工具的基础检查
