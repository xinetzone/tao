# 全局核心规则

本文档定义 AgentForge 仓库中 AI 智能体必须遵守的全局核心原则。这些原则适用于所有工作区（chaos 和 rebirth），是 AGENTS.md 全局契约的核心约束。

## 1. 沟通语言

必须使用中文与用户交流。

## 2. 按需读取

执行特定领域任务前，只读取与当前任务直接相关的规范文件。不得在每次任务中全量加载所有规则和文档。

## 3. 上下文节省

默认遵循"先搜索、再精读、只保留相关上下文"原则。详细策略见 [`.agents/rules/context-economy.md`](context-economy.md)。

## 4. 代码修改

遵循"约定优于配置"，优先参考现有代码风格和项目架构。修改代码前应先读取目标文件的相邻上下文，确认风格一致后再进行修改。

## 5. Python 环境管理

统一使用 `uv` 管理 Python 依赖与虚拟环境。详细规范见 [`.agents/rules/python.md`](python.md)。

## 6. Mermaid 优先

流程、架构、关系、职责映射等可视化逻辑内容，优先使用 Mermaid 基础语法表达。

## 7. 哲学驱动

仅 AgentForge 世界适用：新增核心开发目标遵循"极致简约、大道至简"，以 Ψ=Ψ(Ψ) 为第一性原理。

## 8. 落地导向

新增设计与实现应回答"如何转化为可执行机制、技术方案与业务场景价值"。
