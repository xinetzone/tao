# 帛书老子注读

![封面](images/cover.png)

本目录收录《帛书老子注读》（秦波 著）全本电子版的 Markdown 转录，依帛书本《老子》体例呈现：**德经在前、道经在后**，共 81 章，并附帛书《老子》注音版。

- 底本：长沙马王堆帛书整理小组校勘的帛书《老子》合订本
- 参校：中华书局《帛书老子校注》（高明）、王弼通行本、河上公注本
- 章节次序遵循帛书本，部分位置错乱章（如今 24、22、23、80、81、67-79）已按帛书经文次序回正

## 篇章统计

- 德经：**44 章**（对应通行本第 38-79 章，含次序变化）
- 道经：**37 章**（对应通行本第 1-37 章，含次序变化）
