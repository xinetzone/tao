# AgentForge 规范文档

本目录存放 AgentForge 项目的规范文档（Specifications）。这些文档面向所有读者——人类开发者、AI Agent、工具实现者。

规范文档独立于 `docs/`（人类文档）和 `.agents/docs/`（AI 知识库），是两者的公约数。
