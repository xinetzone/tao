# `jupyterlite`

JupyterLite 支持 Sphinx 扩展：[sphinx-jupyterlite](https://jupyterlite-sphinx.readthedocs.io/en/latest)

[jupyterlite-xeus](https://jupyterlite-xeus.readthedocs.io/en/stable/index.html) 是 JupyterLite 的扩展，它支持完整客户端的 Jupyter 环境，这些环境由编译成 WebAssembly (Wasm) 的 Xeus 内核驱动。它允许用户创建具有自定义预构建环境的静态 Jupyter 部署——无需服务器。

支持的内核列表（并非全部）：

- [xeus-python](https://github.com/jupyter-xeus/xeus-python) | [xeus-python docs](https://jupyterlite-xeus.readthedocs.io/en/stable/)
- [xeus-cpp](https://github.com/compiler-research/xeus-cpp) | [xeus-cpp docs](https://xeus-cpp.readthedocs.io/en/latest/) | [xeus-cpp lab](https://compiler-research.org/xeus-cpp-wasm/lab/index.html)
- [xeus-javascript](https://github.com/jupyter-xeus/xeus-javascript) | [xeus-javascript lab](https://jupyter-xeus.github.io/xeus-javascript/lab/index.html?path=xeus-javascript.ipynb)
- [xeus-octave](https://github.com/jupyter-xeus/xeus-octave) | [xeus-octave docs](https://xeus-octave.readthedocs.io/en/latest/)
