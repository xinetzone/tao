# Fabric

[Fabric](https://www.fabfile.org) 是一个高级 Python 库（支持 2.7 和 3.4+ 版本），旨在通过 SSH 远程执行 shell 命令，并返回有用的 Python 对象。它基于 [Invoke](https://pyinvoke.org/) （子进程命令执行和命令行功能）和 [Paramiko](https://paramiko.org/) （SSH 协议实现）构建，扩展了它们的 API 以相互补充并提供额外的功能。
