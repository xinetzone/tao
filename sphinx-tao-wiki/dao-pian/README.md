## 道的启示

Sphinx 之道，在于：

- **以文为基** - reStructuredText 或 Markdown 作为基础
- **以境为枢** - BuildEnvironment 作为核心枢纽
- **以器为用** - Builder 作为输出工具
- **以域为界** - Domain 作为内容领域划分

> 道冲而用之，又弗盈也。渊呵，似万物之宗。挫其锐，解其纷，和其光，同其尘。湛呵，似或存。吾不知其谁之子，象帝之先。
