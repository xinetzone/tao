# 第一章：始源之道 - Sphinx 起源与定位

> 天下皆知美之为美，恶已。皆知善，斯不善矣。有无之相生也，难易之相成也，长短之相形也，高下之相盈也，音声之相和也，前后之相随，恒也。

## 1.1 Sphinx 之始

Sphinx 诞生于 Python 社区，最初为 Python 官方文档而生，
文档系统。其名取自古埃及的斯芬克斯，象征着智慧与谜题的化身。

> 上善若水。水善利万物而有静，居众人之所恶，故几于道矣。

Sphinx 正如水一般，默默滋养着文档世界，居于众人所不为之地，故近于道。

## 1.2 Sphinx 定位

Sphinx 是一个**智能文档生成器，其特点：

- **以 reStructuredText 为源
- **以多格式输出为用
- **以交叉引用为妙
- **以代码高亮为华
- **以扩展生态为基

## 1.3 Sphinx 愿景

让文档如涓流不息，生生不息，
让知识如江河奔涌，川流不息。

> 持而盈之，不如其已。揣而棁之，不可长保也。金玉盈室，莫之守也。贵富而骄，自遗咎也。功遂身退，天之道也。

**本章要义**：Sphinx 始于文档生成之道，在于默默服务，功成身退。
